# DL_template_copy
Script for copying CPT and lab data (from GIR tool) to Excel DL_template. Create separate Excel sheets for each parameter per unit or per location. Design lines should be added manually.
The following folders should be inside before running. 1) Lab_data_per_units, 2) design_lines_from_template
"design_lines_from_template" folder is where the output Excel sheets are saved
"Lab_data_per_units" folder should include unit-wise Excel sheets for lab data, extracted from the GIR tool

Main inputs
1.DL_input_file.xlsx        -  Excel file containing input data
2.Su_DL_template.xlsx      – Excel template (change per parameter)
From GIR tool: 
3.All_CPTs.xlsx – cpt correlation data for all the locations 
4.Lab_data_per_unit.xlsx – lab data per units (inside "Lab_data_per_units" folder)
